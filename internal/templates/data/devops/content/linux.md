# Linux の基礎

DevOps のすべての土台となるのが Linux の知識です。

## 最低限覚えるべきコマンド

```bash
# ファイル操作
ls, cd, cp, mv, rm, mkdir, find

# テキスト処理
cat, grep, awk, sed, sort, uniq, wc

# プロセス管理
ps, top, kill, nohup

# ネットワーク
curl, wget, netstat, ss, dig, nslookup
```

## 学習のコツ

1. 実際に Linux 環境 (WSL や VM) を用意して手を動かす
2. シェルスクリプトで日常タスクを自動化してみる
3. `man` コマンドでヘルプを読む習慣をつける
